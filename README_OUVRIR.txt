Budget Ajoub - Instructions d'ouverture
======================================

1) Ouvrir le dossier avec Cursor
- Ouvrez Cursor
- File > Open Folder
- Choisissez ce dossier : C:\Users\hp\Desktop\budget ajoub

2) Installer les dependances (si necessaire)
- Ouvrez un terminal dans le projet
- Lancez : npm install

3) Lancer l'application
- Lancez : npm run dev

4) Ouvrir dans le navigateur
- Suivez l'URL affichee dans le terminal (en general http://localhost:5173 pour le web)
- Si l'API est separee, demarrez aussi le service API selon le script du projet.
